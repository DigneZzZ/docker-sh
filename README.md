# Install Guide:

1. Clone the git and edit install-script
```bash
git clone https://github.com/DigneZzZ/docker-sh.git
```
2. Make CHMOD
```bash
chmod +x docker-sh/docker-ce-yml.sh
```
3. Run
```bash
sudo ./docker-sh/docker-ce-yml.sh
```
